# Udemy Auto Translate

Udemyの練習問題（クイズ）で問題が切り替わるたびに、Chromeの翻訳を自動で再トリガーするChrome拡張機能です。

## 何ができるの？

Udemyの英語コースで練習問題を解いているとき、問題が次に進むたびにChromeの翻訳が解除されて毎回やり直す必要がありますよね。この拡張機能はそれを自動化します。

- 問題の切り替わりを自動検知
- Chromeの翻訳を自動で再トリガー
- ポップアップから有効/無効の切り替え可能
- 翻訳先言語の変更も可能（デフォルト: 日本語）

## インストール方法

1. このリポジトリをクローン

```bash
git clone https://github.com/don-yamada/udemy-auto-translate.git
```

2. Chromeで `chrome://extensions` を開く
3. 右上の「デベロッパーモード」をONにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. クローンした `udemy-auto-translate` フォルダを選択

これだけで使えます。

## 使い方

1. Udemyの練習問題ページを開く
2. Chromeの翻訳を一度有効にする（アドレスバーの翻訳アイコン）
3. あとは問題が切り替わるたびに自動で翻訳が再実行されます

ツールバーのアイコンをクリックすると:
- 自動翻訳のON/OFF切り替え
- 翻訳先言語の変更

ができます。

## 注意事項

- この拡張機能はChromeの翻訳機能を再トリガーしているだけで、Udemyのコンテンツの改変・保存・再配布は一切行いません
- 利用は自己責任でお願いします
- Udemyの仕様変更により動作しなくなる可能性があります

## 開発

```bash
cd udemy-auto-translate
npm install
npm test
```
